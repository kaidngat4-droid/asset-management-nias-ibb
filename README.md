## 📌 About the Project
Electronic Asset Management System designed to help institutions manage and track their assets efficiently.

The system allows recording assets, managing depreciation, storing asset images, generating barcodes, exporting reports to Excel, and creating backups.

---

## ⚙️ Technologies Used

- HTML5
- CSS3
- JavaScript
- IndexedDB
- ExcelJS
- JsBarcode

---

## ✨ Features

- 🔐 Secure Login System
- 🆔 Automatic Asset ID Generation
- 📦 Asset Management
- 🖼 Upload Asset Images
- 🏷 Automatic Barcode Generation
- 📊 Excel Export with Professional Formatting
- 🔎 Asset Search
- 💾 Backup System
- 🗑 Clear All Data Option
- 📱 Mobile Friendly Design
